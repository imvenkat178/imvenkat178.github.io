Projects

Cab Booking Application

This project aims to detect cabs close to the user and assign them accordingly using real-time geolocation data from both the user and the driver. Developed with Spring Boot and Kafka, it stores and continuously shares real-time geolocation information between users and cab drivers. The inclusion of Spring Security enhances the robustness of the web application, while REST API facilitates seamless communication between the server and the user.

Toll Bill Generator

This project generates toll bills for highway users. The frontend uses Angular for a dynamic interface, while the backend is built with Spring Boot for robust server-side architecture. Hibernate handles ORM for database interaction, and REST APIs ensure real-time communication and transactions, resulting in a streamlined toll billing process.
